Welcome to my simplified clone of Sociobytes, inspired by the hard work of Aashish Chapain. I encourage you to explore the original project as well.

Tech Stack:

  • Django
  • Postgresql
  • htmx for an enhanced SPA (Single Page Application) feel
  • NiceAdmin theme by BootstrapMade

While the majority of features are operational, it's important to note that this project is still a work in progress. Although not intended as a competitor to Sociobytes, I am committed to ongoing updates and refining the application as time permits. Your feedback and suggestions are welcome as I strive to enhance and polish the user experience.

The source for this project will be available on GitHub. Link will be in the footer once it has been uploaded.
